## Audio → Visual Mappings (quick reference)

- Pitch → vertical position (high = high)
- Loudness → brightness / bloom intensity
- Timbre brightness → edge sharpness / sparkle
- Noise amount → grain density / fog thickness
- Harmonicity → clean banding vs smeared clouds
- Transients → thin vertical streaks / needle highlights
- Reverb → ghost trails / depth haze / temporal smear
- Distortion → texture thickness + edge fuzz + interference shimmer
- Chorus/flange/phaser → phase drift ripples + moiré shimmer
- Compression → flattened contrast + "pumped" glow rhythm

Rule: interference must live INSIDE haze, not on top as crisp UI.
